After the End - Deposed - Official Unofficial 4chan Fan Fork Fork (AtEDOU4cFFF) (Name Pending)

=================================================================================================
============================================IMPORTANT============================================
=================================================================================================

Otherwise known as AtE Deposed or just Deposed, this mod is for Crusader Kings II and can be 
downloaded from its Github page:
https://github.com/DeposedAnon/After-the-End-Deposed

Deposed is a SUBMOD for After the End Fan Fork and REQUIRES that mod to run properly.

=================================================================================================
============================================IMPORTANT============================================
=================================================================================================

After the End Fan Fork can be downloaded from:
Its Github page:
https://github.com/9Kbits/ateff
Or its Steam Workshop page:
https://steamcommunity.com/sharedfiles/filedetails/?id=1341828311

The Github version of AtEFF is the preferred version to use for Deposed, as it's the version that
Deposed is based on. It's also slightly more updated than the version on the Steam Workshop.

=================================================================================================
==============================================ABOUT==============================================
=================================================================================================

Deposed was started by anons on the /vst/ board of 4chan to restore content that was previously 
removed or changed by the Fan Fork Dev team. It also fixes numerous bugs, and recently has 
started adding in new content of its own.

It's planned for Deposed to become a standalone mod in the future, but there's no set date for 
that yet.
